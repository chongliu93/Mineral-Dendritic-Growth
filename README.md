## Overview
This repository contains the datasets supporting the following work:

- Xinshuo Huang, Chong Liu, Victor M. Calo, Klaus Regenauer Lieb, Manman Hu, Noreen J. Evans, Kezhang Qin and Mingjian Cao. **"Fractal morphology of mineral dendrites reveals ore-forming conditions"**.
